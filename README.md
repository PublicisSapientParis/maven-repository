# maven-repository

Maven repository
```xml
<repository>
    <id>PublicisSapientParis.github.io.snapshots</id>
    <name>PublicisSapientParis.github.io maven-repository</name>
    <url>https://raw.githubusercontent.com/PublicisSapientParis/maven-repository/master/snapshots/</url>
</repository>
<repository>
    <id>PublicisSapientParis.github.io.releases</id>
    <name>PublicisSapientParis.github.io maven-repository</name>
    <url>https://raw.githubusercontent.com/PublicisSapientParis/maven-repository/master/releases/</url>
</repository>
```
for plugins
```xml
<pluginRepository>
    <id>PublicisSapientParis.github.io</id>
    <name>PublicisSapientParis.github.io maven-repository</name>
    <url>https://raw.githubusercontent.com/PublicisSapientParis/maven-repository/master/releases/</url>
</pluginRepository>

```
